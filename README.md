# Customer Churn Prediction & Analysis

## Overview
Telecom company ke 7043 customers ka data analyze karke churn prediction model banaya.

## Key Findings
- 26.6% customers churn karte hain
- Month-to-month contract wale customers mein 43% churn rate hai
- TotalCharges aur MonthlyCharges sabse bade churn factors hain
- Naye customers (0-10 months) mein churn rate sabse zyada hai

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Models Used
- Random Forest — 79.89% accuracy
- Logistic Regression — 78.68%
- Gradient Boosting — 79.18%

## Dataset
IBM Telco Customer Churn Dataset (Kaggle)
